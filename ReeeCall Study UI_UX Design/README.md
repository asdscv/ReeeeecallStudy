# ReeeCall Study - 플래시카드 학습 앱

깔끔하고 미니멀한 UI의 1인용 플래시카드 학습 웹앱입니다. SRS(간격 반복 학습) 알고리즘을 기반으로 효율적인 학습을 지원합니다.

## 주요 기능

### ✅ 구현된 기능
- **이메일 매직 링크 인증** - 비밀번호 없이 간편하게 로그인
- **덱 관리** - 색상과 아이콘으로 구분되는 학습 덱 생성/수정
- **카드 관리** - 앞면/뒷면 형식의 플래시카드 추가/편집
- **SRS 학습 시스템** - SM-2 알고리즘 기반 간격 반복 학습
- **대시보드** - 학습 통계 및 GitHub 스타일 히트맵
- **템플릿 시스템** - 커스터마이즈 가능한 카드 필드
- **반응형 디자인** - PC 우선, 모바일 지원

### 학습 모드
1. 🧠 **SRS (간격 반복)** - 잊을 때쯤 자동으로 복습
2. 🔄 **순차 복습** - 새 카드 + 처음부터 복습
3. 🎲 **랜덤** - 무작위로 섞어서 학습
4. ➡️ **순서대로** - 첫 카드부터 순서대로

## 🚨 중요: 이메일 설정 필요

이 앱은 Supabase 이메일 인증을 사용합니다. 로그인이 작동하려면 다음 설정이 필요합니다:

### Supabase 이메일 설정 방법

1. **Supabase 대시보드 접속**
   - https://supabase.com/dashboard 로 이동
   - 프로젝트 선택

2. **이메일 템플릿 활성화**
   - 좌측 메뉴에서 `Authentication` > `Email Templates` 선택
   - `Magic Link` 템플릿 확인
   - 이메일이 발송되는지 테스트

3. **개발 환경에서 테스트**
   - Supabase는 기본적으로 개발 환경에서 이메일을 전송합니다
   - 실제 이메일 주소로 테스트해보세요

4. **프로덕션 설정 (선택사항)**
   - SMTP 설정을 통해 자체 이메일 서버 사용 가능
   - `Settings` > `Auth` > `SMTP Settings`

## 사용 방법

### 1. 첫 로그인
1. 이메일 주소 입력
2. 받은 이메일의 링크 클릭
3. 자동으로 로그인됩니다

### 2. 덱 만들기
1. "덱 관리" 탭으로 이동
2. "새 덱 만들기" 버튼 클릭
3. 덱 이름, 색상, 아이콘 선택

### 3. 카드 추가
1. 생성한 덱 클릭
2. "카드 추가" 버튼 클릭
3. 앞면/뒷면 내용 입력

### 4. 학습 시작
1. "학습 시작" 버튼 클릭
2. 학습 모드 선택
3. 카드를 클릭하여 뒤집기
4. Again/Hard/Good/Easy로 평가

## 기술 스택

- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS v4
- **Router**: React Router v7
- **Backend**: Supabase (Edge Functions + Hono)
- **Database**: Supabase KV Store
- **Auth**: Supabase Auth (Magic Link)
- **Animation**: Motion (Framer Motion)

## 데이터 구조

### Deck (덱)
- 이름, 설명, 색상, 아이콘
- 사용할 템플릿 ID

### Card (카드)
- 필드 데이터 (앞면/뒷면 등)
- SRS 데이터 (interval, easeFactor, nextReview)
- 상태 (new, learning, review, suspended)

### Template (템플릿)
- 필드 정의 (이름, 타입)
- 앞면/뒷면 레이아웃 설정

## SRS 알고리즘

SM-2 알고리즘 기반:

- **Again (1)**: 10분 후 다시
- **Hard (2)**: 현재 간격 × 1.2
- **Good (3)**: 현재 간격 × easeFactor
- **Easy (4)**: 현재 간격 × easeFactor × 1.3
