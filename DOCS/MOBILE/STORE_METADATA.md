## 긴 설명 (양쪽 공통)
ReeeeecallStudy is an intelligent flashcard application that helps you memorize anything effectively using spaced repetition science (SRS).

**Smart Study Modes**
• SRS Review — Scientifically optimized review scheduling
• Sequential Review — Go through cards in order
• Random — Shuffle and study
• Cramming — Rapid-fire until mastered
• By Date — Study cards by upload date

**Powerful Features**
• Create unlimited decks with customizable card templates
• Import/Export cards via CSV
• Track your progress with detailed study statistics
• Daily study streaks and mastery tracking
• Heatmap visualization of your study activity
• AI-powered card generation (bring your own API key)
• Text-to-Speech for pronunciation practice

**Marketplace**
• Browse and discover shared decks from the community
• Publish your own decks to help others learn

**Cross-Platform**
• Web app + iOS + Android
• Your data syncs seamlessly across all devices

**Privacy First**
• Your data is yours — export anytime
• AI API keys encrypted locally, never stored on servers
• Account deletion available in settings

Start learning smarter today with ReeeeecallStudy!
