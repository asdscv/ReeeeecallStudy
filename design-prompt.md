# ReeeCall Study — UI/UX 디자인 프롬프트

## 앱 개요

"ReeeCall Study"는 1인용 플래시카드 학습 웹앱입니다. Anki와 유사하지만 더 깔끔한 UI를 목표로 합니다.
기술 스택: React + TailwindCSS. PC 우선, 모바일 반응형.

---

## 디자인 시스템

### 브랜딩
- 앱 이름: **ReeeCall Study**
- 성격: 깔끔, 미니멀, 집중력 있는 학습 도구
- 톤: 차분하고 전문적, 불필요한 장식 없음

### 색상 팔레트
- Primary: Blue (#3B82F6) — 버튼, 링크, 액센트
- Background: Gray-50 (#F9FAFB) — 전체 배경
- Surface: White (#FFFFFF) — 카드, 모달, 패널
- Text Primary: Gray-900 (#111827)
- Text Secondary: Gray-500 (#6B7280)
- Text Muted: Gray-400 (#9CA3AF)
- Border: Gray-200 (#E5E7EB)
- Success: Green-500 (#22C55E)
- Warning: Amber-500 (#F59E0B)
- Error: Red-500 (#EF4444)
- SRS 등급별 색상:
  - Again: Red-500
  - Hard: Amber-500
  - Good: Blue-500
  - Easy: Green-500

### 타이포그래피
- 페이지 제목: 2xl (1.5rem), bold
- 섹션 제목: lg (1.125rem), semibold
- 본문: sm~base (0.875~1rem), normal
- 보조 텍스트: sm (0.875rem), gray-500
- 카드 학습 시 메인 텍스트: 4xl~5xl (2.25~3rem), bold

### 컴포넌트 스타일
- 카드/패널: bg-white, rounded-xl, border border-gray-200, shadow 없거나 shadow-sm
- 버튼 (Primary): bg-blue-600, text-white, rounded-lg, hover:bg-blue-700
- 버튼 (Secondary): bg-white, border border-gray-300, text-gray-700, rounded-lg
- 버튼 (Danger): bg-red-50, text-red-600, hover:bg-red-100
- Input: border border-gray-300, rounded-lg, px-4 py-3, focus:border-blue-500, focus:ring-2 ring-blue-500/20
- Badge: px-2 py-0.5, rounded-full, text-xs, font-medium

### 레이아웃
- 최대 너비: max-w-6xl (1152px), 중앙 정렬
- 헤더: 높이 56px, sticky top-0, bg-white, border-b
- 사이드바 없음 — 상단 네비게이션 방식
- 콘텐츠 영역: px-4 py-6

## 페이지별 디자인

### 1. 로그인 페이지 (/auth/login)

**레이아웃:** 화면 정중앙에 카드 1개

**구성:**
- 앱 로고/이름: "ReeeCall Study" (상단 중앙, 2xl bold)
- 부제: "이메일로 간편 로그인" (gray-500)
- 이메일 입력 필드 1개 (placeholder: "your@email.com")
- 파란 버튼: "로그인 링크 보내기" (전체 너비)
- 하단 안내문: "비밀번호 없이 이메일 링크로 안전하게 로그인합니다." (xs, gray-400)
- 비밀번호 필드는 **없음**

**발송 완료 상태:**
- 같은 카드에서 내용만 변경
- 📧 이모지 (5xl)
- "이메일을 확인해주세요" (xl bold)
- "{email}로 로그인 링크를 보냈습니다." (gray-500)
- "다른 이메일로 다시 보내기" 텍스트 링크

### 2. 헤더/네비게이션

**구성:** 가로 1줄 헤더
- 좌측: "ReeeCall Study" 로고/텍스트 (lg bold)
- 중앙: 네비게이션 탭 4개
  - 📊 대시보드
  - 📚 덱 관리
  - 📋 템플릿
  - ⚙️ 설정
- 활성 탭: bg-blue-50, text-blue-700, font-medium
- 비활성 탭: text-gray-600, hover:bg-gray-100
- 우측: 사용자 이메일 (sm gray-500) + "로그아웃" 텍스트 버튼

---

### 3. 대시보드 (/)

**레이아웃:** 상단 요약 카드 → 잔디 히트맵 → 그래프

**상단 요약 (4열 그리드):**
- 오늘 학습한 카드 수
- 전체 카드 수
- 복습 예정 카드 수
- 연속 학습 일수 (streak)
- 각 카드: 아이콘 + 숫자(2xl bold) + 라벨(sm gray-500)

**잔디 히트맵:**
- GitHub Contribution 스타일 365일 격자
- 색상: gray-100(0) → green-200 → green-400 → green-600 → green-800
- 마우스 오버 시 툴팁: "2026-02-14: 45장 학습"
- 하단에 범례: 적음 ■■■■■ 많음

**이번 주 복습 예정 (미니 바차트):**
- 7일 가로 바 차트
- 각 바에 날짜 + 카드 수

**최근 덱 바로가기:**
- 가장 최근 학습한 덱 3개를 가로 카드로 표시
- 각 카드: 덱 아이콘+이름, "복습 N개", "학습 시작" 버튼

---

### 4. 덱 목록 (/decks)

**상단:** "내 덱" 제목 + "새 덱 만들기" 파란 버튼

**덱 카드 그리드 (2~3열):**
각 덱 카드:
- 좌측에 색상 바 (덱 color, 세로 4px 또는 좌측 border)
- 덱 아이콘 + 이름 (lg semibold)
- 총 카드 수: "152장"
- SRS 요약: "새 카드 12 · 복습 예정 34"
- 마지막 학습: "2시간 전"
- 우측 상단: ··· 더보기 메뉴 (수정, 보관, 삭제)
- 하단: "학습 시작" 버튼

**빈 상태:**
- 일러스트 또는 큰 아이콘
- "아직 덱이 없습니다. 새 덱을 만들어보세요."

### 5. 덱 생성/수정 모달

**모달 크기:** max-w-md

**필드:**
- 덱 이름 (필수, 텍스트 입력)
- 설명 (선택, textarea)
- 색상 선택: 8가지 원형 색상 칩 (클릭으로 선택)
  - #3B82F6, #EF4444, #22C55E, #F59E0B, #8B5CF6, #EC4899, #06B6D4, #6B7280
- 아이콘 선택: 이모지 8개 중 선택
  - 📚 📖 🇨🇳 🇺🇸 🇯🇵 🧠 💡 📝
- 기본 템플릿 선택: 드롭다운 (사용자의 템플릿 목록)
- 하단: "취소" + "저장" 버튼

### 6. 덱 상세 (/decks/:id)

**상단 영역:**
- 덱 아이콘 + 이름 (2xl bold)
- 덱 설명 (gray-500)
- 통계 배지: 전체 N장 · 새 카드 N · 복습 예정 N
- 액션 버튼 행: "학습 시작" (Primary) · "카드 추가" (Secondary) · "가져오기" · "내보내기"

**탭:**
- [카드 목록] [업로드 일자] [통계]

**카드 목록 탭:**
- 검색바 + 필터 드롭다운 (상태별, 태그별)
- 테이블 뷰:
  - 체크박스 | 필드1 (앞면) | 필드2 (뒷면) | 상태 배지 | 다음 복습일 | 추가일
  - 상태 배지 색상: new=blue, learning=amber, review=green, suspended=gray
- 하단: 선택된 항목 일괄 작업 (삭제, 상태 변경)

**업로드 일자 탭:**
- 날짜별 그룹: "2026-02-14 (32장)" → 클릭하면 해당 날짜 카드 표시
- 각 날짜에 "이 날짜 카드만 학습" 버튼

---

### 7. 카드 편집 모달

**모달 크기:** max-w-lg

**구성:**
- 상단: "카드 편집" 또는 "새 카드 추가"
- 템플릿에 정의된 필드가 동적으로 렌더링:
  - text 필드: 라벨(필드 이름, sm semibold) + 텍스트 입력
  - image 필드: 라벨 + 파일 드래그&드롭 영역 (점선 테두리)
  - audio 필드: 라벨 + 파일 업로드 버튼
- 태그 입력: 칩 형태로 표시, 입력 후 Enter로 추가
- 하단: "취소" + "저장" 버튼

---

### 8. 카드 템플릿 목록 (/templates)

**상단:** "카드 템플릿" 제목 + "새 템플릿" 파란 버튼

**템플릿 카드 (세로 목록):**
각 항목:
- 템플릿 이름 (semibold)
- 필드 수: "필드 5개"
- 필드 목록 미리보기: "한자 · 뜻 · 병음 · 예문 · 오디오" (sm gray-500)
- 기본 템플릿 배지: "기본" (blue badge)
- 우측: "편집" 버튼

---

### 9. 템플릿 편집 페이지 (/templates/:id/edit)

**레이아웃:** 2단 구성

**좌측: 필드 관리**
- "필드 관리" 섹션 제목 + "(최대 10개, 현재 N개)"
- 필드 목록 (드래그 가능):
  - 드래그 핸들 (☰) + 순번 + 필드 이름 + 타입 드롭다운 (text/image/audio) + 삭제 버튼(X)
  - 필드 이름은 인라인 편집 가능
- "필드 추가" 버튼

**우측: 앞면/뒷면 설정 (2열)**
- 각 면:
  - "앞면 설정" / "뒷면 설정" 제목
  - 추가된 필드 목록 (드래그 가능):
    - 필드 이름 + 스타일 드롭다운 (primary/secondary/hint/detail/media)
  - "필드 추가" 드롭다운
  - 구분선
  - 미리보기 영역:
    - 실제 카드처럼 렌더링 (연한 배경, rounded-xl)
    - 샘플 데이터로 표시

**하단:** "저장" 버튼

---

### 10. 학습 모드 선택 모달

**모달 크기:** max-w-sm

**구성:**
- "학습 모드 선택" 제목
- 4개 모드 카드 (세로 쌓기, 클릭으로 선택):
  - 🧠 SRS (간격 반복) — "잊을 때쯤 알아서 복습"
  - 🔄 순차 복습 — "새 카드 + 처음부터 복습"
  - 🎲 랜덤 — "무작위로 섞어서 학습"
  - ➡️ 순서대로 — "첫 카드부터 순서대로"
- 선택된 모드: border-blue-500, bg-blue-50
- 구분선
- 필터 옵션:
  - 📅 업로드 일자: 드롭다운 (전체 / 날짜 선택)
  - 🔢 학습할 카드 수: 숫자 입력 (기본 50)
  - 순차 복습 모드 선택 시: 새 카드 배치 크기 / 복습 배치 크기 추가 표시
- "학습 시작" 파란 버튼

---

### 11. 학습 세션 화면 (/study/:deckId)

**레이아웃:** 전체 화면, 헤더 숨김, 집중 모드

**상단 프로그레스 바:**
- sticky 상단, 높이 4px
- 색상 그라데이션: 0~50% red → 50~80% amber → 80~100% green
- 바 위에: "23 / 50" 텍스트 + 현재 모드 아이콘 + "X" 종료 버튼

**카드 영역 (화면 중앙):**
- 큰 카드 (max-w-2xl, 최소 높이 400px)
- bg-white, rounded-2xl, shadow-lg, border
- 카드 내용은 템플릿 레이아웃에 따라 렌더링:
  - primary 스타일: text-4xl~5xl, font-bold, text-center
  - secondary 스타일: text-2xl, text-center
  - hint 스타일: text-xl, text-gray-400, text-center
  - detail 스타일: text-base, text-gray-500, mt-4
  - media 스타일: 이미지 max-w-xs mx-auto / 오디오 재생 버튼

**앞면 상태:**
- 카드에 앞면 필드만 표시
- 하단: "답 보기" 버튼 또는 "Space로 뒤집기" 힌트 (gray-400)

**뒷면 상태 (뒤집은 후):**
- 카드 상단: 앞면 내용 (작게, gray-400)
- 카드 중앙: 뒷면 필드들 (템플릿 레이아웃대로)
- 카드 하단 아이콘: 🔊 TTS 재생 버튼 + ✏️ 편집 버튼

**평가 버튼 (SRS 모드, 카드 아래):**
- 4개 버튼 가로 배치, 같은 너비:
  - Again (1): bg-red-500, text-white, "10분"
  - Hard (2): bg-amber-500, text-white, "1일"
  - Good (3): bg-blue-500, text-white, "3일"
  - Easy (4): bg-green-500, text-white, "7일"
- 각 버튼에 예상 다음 복습일 표시

**평가 버튼 (비SRS 모드, 카드 아래):**
- 2개 버튼:
  - "모르겠다" (bg-red-100, text-red-700)
  - "알겠다" (bg-green-100, text-green-700)
- 또는 순서대로 모드: "다음 →" 버튼 1개

**뒤집기 애니메이션:**
- CSS 3D flip: rotateY(180deg), transition 0.4s

### 12. 학습 완료 화면

**레이아웃:** 화면 중앙

**구성:**
- 큰 체크마크 또는 축하 아이콘
- "학습 완료!" (2xl bold)
- 요약 통계 카드:
  - SRS 모드: Again/Hard/Good/Easy 각각 수, 정답률
  - 순차 복습: "새 카드 100개 + 복습 150개 완료"
  - 기타: "50개 카드 학습 완료"
- 걸린 시간, 평균 응답 시간
- "대시보드로" 버튼 + "한 번 더 학습" 버튼

### 13. 설정 페이지 (/settings)

**레이아웃:** 섹션별 카드

**학습 설정:**
- 일일 신규 카드 한도: 슬라이더 (5~200, 현재 값 표시)
- 기본 학습 모드: 4개 라디오 버튼
- 타임존: 드롭다운

**TTS 설정:**
- TTS ON/OFF: 토글 스위치
- TTS 언어: 드롭다운 (zh-CN, en-US, ko-KR, ja-JP)
- TTS 엔진: 라디오 (Web Speech API / edge-tts)

**테마:**
- 3개 선택지: 라이트 / 다크 / 시스템

**계정:**
- 현재 이메일 표시
- 로그아웃 버튼

---

### 14. 가져오기/내보내기 모달

**가져오기:**
- 파일 드래그&드롭 영역 (큰 점선 박스)
- 지원 포맷: JSON, CSV
- 업로드 후 미리보기 테이블 (5행)
- CSV인 경우: 헤더 → 필드 매핑 드롭다운
- 중복 처리: 라디오 (건너뛰기 / 덮어쓰기 / 새로 추가)
- "가져오기 실행" 버튼

**내보내기:**
- 포맷 선택: JSON / CSV 라디오
- "내보내기" 버튼 → 즉시 다운로드
